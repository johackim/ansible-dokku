Ansible-dokku [![Ansible Galaxy](https://img.shields.io/badge/galaxy-dokku-660198.svg)][1]
===

> Install [dokku][2] on debian servers.

Requirements
---

- Ansible >= 2.2.1

Example Playbook
---

```
- hosts: localhost
  roles:
    - dokku
```
